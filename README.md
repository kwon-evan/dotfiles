# DOTFILES

~/.config  
　　├── starship.toml  
　　├── .tmux.conf  
　　├── setup.sh  
　　├── yazi/  
　　├── wezterm/  
　　└── nvim/  

## Requirements

- Neovim > 0.9.0
- [Nerd Font](https://www.nerdfonts.com/)
- [ripgrep](https://github.com/BurntSushi/ripgrep)
- npm
- treesitter
- fzf

## Installation

```bash
git clone https://github.com/kwon-evan/dotfiles.git ~/dotfiles/ --depth 1 && ~/dotfiles/setup.sh
```
